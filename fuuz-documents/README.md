# fuuz-documents

Fuuz Document Designer skill — design and build document templates, reports, and dashboards using Stimulsoft Reports within the Fuuz platform.

## What This Skill Covers

- Stimulsoft Reports & Dashboards integration with Fuuz
- Band-based layout system (20+ band types)
- Expression syntax for calculated fields and conditionals
- Data binding from Fuuz models to report templates
- Component types (text, images, charts, barcodes, tables, cross-tabs, gauges)
- Common manufacturing document patterns (travelers, pick lists, inspection reports, labels)
- Document rendering and export (PDF, Excel, Word, HTML)
