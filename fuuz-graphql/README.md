# fuuz-graphql

Fuuz GraphQL skill — write, debug, and optimize GraphQL queries and mutations for the Fuuz Industrial Operations Platform.

## What This Skill Covers

- How Fuuz auto-generates GraphQL APIs from data models
- Application API vs System API scopes
- Query syntax (where, orderBy, take, skip, edges/node pattern)
- Mutation syntax (create, update, delete, batch payload arrays)
- Aggregation queries for telemetry and production data
- Relationship traversal and nested queries
- GraphQL patterns for data flows (query/mutate nodes)
- GraphQL patterns for screen elements (data providers, tables, forms)
